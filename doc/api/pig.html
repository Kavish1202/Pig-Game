pig.turn module¶

Turn logic for the Pig game.

class pig.turn.Turn(player: Player, dice: Dice | None = None)[source]¶

Bases: object

Handles one player’s turn in the Pig game.

Keeps track of rolls, points earned this turn, and whether the turn has ended.

hold() → None[source]¶

End the turn and add the turn points to the player’s score.

Does nothing if the turn has already ended.

reset() → None[source]¶

Reset the turn so it can be reused.

roll() → int[source]¶

Roll the dice once.

If it’s a 1, the turn ends and all points are lost. Otherwise, the roll value is added to this turn’s total.
